Procnanny:
Procnanny monitors running processes and kills them if they run for longer than a 
specified amount of time.  Procnanny takes one input, which is a file with the max
number of seconds a monitored program should run for as its first line, and the names
of the processes to be monitored on the following lines.  Information is written to
a log file, whose location is specified by the enviornmental variable PROCNANNYLOGS. 
If specifying the logfile enviornment variable as a full path, it must be started 
with "/home/user" (using "~" will not work).

An executable called "procnanny" is generated by calling "make".  This executable can
be removed by calling "make clean".  The program can be run by calling "./procnanny 
input_file_name". Or, use "export PATH=$PATH:." then "procnanny input_file_name" (this
removes the "./" from the function call).
